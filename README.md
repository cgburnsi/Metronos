# Metronos

**Metronos** (from the Ancient Greek *μέτρον* /metron/, meaning "measure" or "that by which anything is measured.")

### The Philosophy

* **Lean:** Small footprint, no heavy dependencies.
* **Dynamic:** No type-hint clutter. It works the way Python was intended to work.
* **Intuitive:** It handles the dimensional math so you can focus on the physics.

---

### Why this works:

* **The Etymology:** Putting the Greek definition right at the top justifies the "fancy" name and immediately signals that this is a library about measurement.
* **The Contrast:** By mentioning Pint immediately, you’re doing the reader a favor. You're saying, "If you want the giant powerhouse, go over there; if you want something light, stay here."
* **The Tone:** It sounds like a developer talking to another developer over a beer, which is usually the most trusted tone on GitHub.
